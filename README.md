# 内容介绍

本项目是一款基于微信平台的健身小助手小程序，通过整合健身知识、计划制定、运动记录等功能，帮助用户养成良好的健身习惯。小程序采用前后端分离的开发模式，前端负责展示界面与交互，后端提供数据支持。通过这个项目，我们希望让更多的人关注健康，积极参与健身运动。

# 核心代码

以下是项目中的一段核心代码，展示了如何通过mybatis实现与数据库的交互：

// 引入Mybatis依赖
import org.apache.ibatis.session.SqlSession;

// 获取SqlSession
SqlSession sqlSession = sqlSessionFactory.openSession();

// 创建一个Mapper接口的代理实例
UserMapper userMapper = sqlSession.getMapper(UserMapper.class);

// 执行查询操作
User user = userMapper.getUserById(1);

// 输出查询结果
System.out.println(user.getName());

// 关闭SqlSession
sqlSession.close();
